# framework/resources 资源说明

本目录保存框架层给项目代理使用的公共模板、提示词和 Claude Code 约束。它不是模块实现目录，也不保存运行状态。

## 内容边界

- `templates/`：新项目、论文草稿、项目工作区的文件骨架。
- `prompts/`：框架层可复用提示词，用于说明任务分工、证据要求和项目代理行为边界。
- `automation/`：可选多代理协议和角色说明；框架必须在没有 subagent 时仍能运行。
- `claude/`：提供给项目 Claude Code 的 agent、command、skill 模板。

## 禁止内容

- 七个模块的私有算法、抓取器、实验执行器或论文编译器实现。
- API key、账号、真实数据集、下载仓库、训练日志和论文 claim 证据。
- Web 前端构建产物或状态缓存。
- 针对单个项目/论文的临时修补提示。

项目 Claude Code 工作时只能修改项目文件夹；本目录只提供公共模板和规约。
