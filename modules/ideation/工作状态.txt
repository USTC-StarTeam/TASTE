# Ideation 模块工作状态

## 本轮较大修改

1. 新增独立生成链路：`main.py --action generate`，可只凭显式论文精读产物、研究主题/兴趣/画像/配置调用 Claude Code 生成新论文 idea。
2. 新增分类脚本目录：

   - `scripts/artifact_io/`：输入读取、证据规范化、ideation 内安全写入。
   - `scripts/claude/`：Claude Code CLI 查找、调用、JSON 解析。
   - `scripts/ideation_quality/`：idea schema、质量审计、Markdown/arena 渲染。
3. 新增 `modules/ideation/.gitignore`，忽略 `runs/`、`tmp/`、`__pycache__/`、`*.pyc` 等运行中间产物。
4. 重建 `script_manifest.json`，登记新增分类脚本。
5. 重写 README，说明独立使用方式、输入输出口径、运行流程和脚本职责。
